majifix priority

## Requirements
- [NodeJS v9.3.0+](https://nodejs.org)

## Usage
- Clone
```sh
$ npm install majifix-priority
```

## Testing
* Clone this repository

* Install all development dependencies
```sh
$ npm install
```

* Then run test
```sh
$ npm test
```
